# 插件中文教程

👋 欢迎来到插件中文教程


## 开始你的旅程

![:PluginsWiki](https://count.kjchmc.cn/get/@:PluginWiki)

在开始前，请确认你拥有以下基础：

- 具备基础的计算机操作技能 - 如解压、新建文件夹、打开文件、安装软件等
- 具备使用搜索引擎搜索关键词的能力
- 拥有一台能够正常游玩 Minecraft 的电脑
- 具有一定的英语基础

如果你确定你符合条件，请点击一侧的目录开始阅读文档。
